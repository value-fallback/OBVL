## Installation

**Via [NPM](https://www.npmjs.com):**

[![NPM](https://nodei.co/npm/obvl.png?downloads=true&downloadRank=true&stars=true)](https://www.npmjs.com/package/obvl)

```
npm install obvl
```

**Via [Yarn](https://yarnpkg.com):**

```
yarn add obvl
```
